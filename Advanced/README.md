# 🧠 Sentiment Analysis using BERT

## 📌 Project Overview
This project implements a BERT-based Language Model for sentiment analysis using Natural Language Processing (NLP). The model predicts whether a given text expresses positive or negative sentiment.

---

## 🧠 Language Model Used
- BERT (DistilBERT Sentiment Analysis Pipeline)

---

## ⚙️ Technologies Used
- Python
- Transformers
- PyTorch
- Pandas
- Matplotlib

---

## 📊 Project Workflow
1. Install and import required libraries
2. Load pretrained BERT model
3. Perform sentiment analysis on text reviews
4. Analyze model predictions
5. Visualize sentiment results
6. Evaluate strengths and limitations of the model

---

## 🔍 Research Questions
- How effectively does BERT understand sentiment?
- Can BERT understand contextual meaning in text?
- Does the model struggle with neutral or mixed opinions?

---

## 📈 Results and Insights
The BERT model successfully predicted positive and negative sentiments with high confidence scores. The project demonstrated the effectiveness of transformer-based language models in NLP tasks.

---

## ⚠️ Limitations
- May struggle with sarcasm or ambiguous statements
- Requires significant computational resources

---

## 🚀 Future Improvements
- Fine-tuning BERT on custom datasets
- Expanding to multilingual sentiment analysis
- Deploying as a web application
