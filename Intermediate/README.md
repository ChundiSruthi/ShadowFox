# 🚗 Car Selling Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts the selling price of cars using Machine Learning techniques. Different car-related features such as fuel type, transmission type, kilometers driven, ownership, and present price were used to train regression models and predict the selling price accurately.

---

## 🧠 Models Used
- Linear Regression
- Random Forest Regressor

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 📊 Dataset Features
- Year
- Present_Price
- Kms_Driven
- Fuel_Type
- Seller_Type
- Transmission
- Owner

Target Variable:
- Selling_Price

---

## 🔄 Data Preprocessing
- Removed unnecessary columns
- Handled categorical data using encoding
- Split dataset into training and testing sets
- Prepared features and target variables

---

## 📈 Model Evaluation
The models were evaluated using:
- RMSE (Root Mean Squared Error)
- R² Score

Random Forest Regressor performed better than Linear Regression with improved prediction accuracy.

---

## 📉 Visualization
Scatter plots were used to compare actual car prices and predicted prices for model performance analysis.

---

## ⚠️ Challenges Faced

## 📌 Conclusion
This project demonstrated how Machine Learning can be used for car price prediction effectively. Among the implemented models, Random Forest Regressor achieved better performance and provided more accurate predictions compared to Linear Regression.
