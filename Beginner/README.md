# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview
This project focuses on predicting house prices using Machine Learning regression models. Various housing-related features such as location, population, total rooms, median income, and households were used to train models and predict median house values accurately.

---

## 🧠 Models Used
- Linear Regression
- Random Forest Regressor

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 📊 Dataset Features
- longitude
- latitude
- housing_median_age
- total_rooms
- total_bedrooms
- population
- households
- median_income

Target Variable:
- median_house_value

---

## 🔄 Data Preprocessing
- Loaded and explored the dataset
- Checked for missing values
- Split dataset into features and target variables
- Performed train-test splitting for model training

---

## 📈 Model Evaluation
The models were evaluated using:
- MSE (Mean Squared Error)
- RMSE (Root Mean Squared Error)
- R² Score

Random Forest Regressor achieved better performance and prediction accuracy compared to Linear Regression.

---

## 📉 Visualization
Scatter plots were used to compare actual house prices and predicted house prices for performance analysis.

## 📌 Conclusion
This project demonstrated how Machine Learning can be used for house price prediction effectively. Among the implemented models, Random Forest Regressor provided better prediction accuracy and overall performance compared to Linear Regression.
